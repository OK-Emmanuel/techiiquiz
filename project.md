### Project Overview: Interactive Well Control Training Platform (Drilling Supervisor & Related Courses)
The goal is to create an online, interactive quiz/study system for accredited courses (primarily Drilling Supervisor certification, with references to OGOR and others). The platform emphasizes mobile-friendly access (smartphones, tablets, PCs) and mirrors the style of official IADC WellSharp exams (e.g., Drilling Operations Supervisor Surface Exam format, viewable at https://iadc.org/accreditation/wellsharp/sample-exams/ — select Sample Exams > Drilling Operations > Supervisor > Surface Exam). Questions should resemble those examples, focusing on multiple-choice, with no need for math input or graphics in quizzes (as those are covered in daily exercises).

### Quizzes & Study Materials
- **Access Control**: Secure, password-protected via student ID and temporary password (auto-generated at checkout).
- **Structure**:
  - Separate daily quiz sets for accredited courses (e.g., Day 1, Day 2, Day 3).
  - Two modes per day/topic:
    - **Study Guide** — 100 questions/answers (Q/A) per set.
      - Interactive learning tool.
      - If incorrect answer selected → system responds: "Incorrect – please select again" (or "please select another answer") → forces retry until correct answer is chosen → then advances to next question.
    - **Practice Test** — Typically 33–35 questions (e.g., 35 Practice Test Questions noted in some sets).
      - Simulates real testing conditions.
      - Student selects answers without immediate feedback.
      - At end of test: Tabulate numerical grade/score.
      - Provide interactive review list of missed questions.
      - In review mode: Incorrect answers marked with an "X"; correct answer highlighted in **bold print**.
- **Content Sources**:
  - Current quiz database maintained in Excel; prefer import functionality to the platform.
  - Specific sets mentioned:
    - Subsea Questions (for Day 3 of Drilling Supervisor's Certification): 100 Study Guide Q/A + 35 Practice Test Q/A.
    - Updates to quiz database provided (including two *.pdf files as Access database output for easier reading).
- **Additional Features**:
  - Optional AI chat to assist students in solving problems (if feasible).
  - Question styling to mirror IADC exam format (professional, clear presentation as seen in official samples).

### Booking & Class Management Page
- Interactive student calendar showing two courses per week (Monday–Thursday):
  - Drilling Supervisor.
  - OGOR.
- Features:
  - View course curriculum details.
  - Check seat availability (maximum class size: 12).
  - Self-booking of classes.
  - Ability to change bookings as needed.
  - Integrated cart and checkout process.
  - Three-month rolling calendar view.
  - Future scalability for multiple training locations.
- Instant access to electronic classroom materials upon booking/confirmation.
- Pre-class recommendation: Review reference material before classes begin (provided at class: Workbook, Pocket Pro Guide, Killsheets, Formula Sheets).
- **Class Rules** (to display clearly):
  - Well Control is a TEAM Effort.
  - Do Not Dominate.
  - Respect Opinions.
  - Respect Time.
  - Must Not Be Absent More Than 4 hours.
  - No Phone Usage During Lectures (must be on silent/vibrate; calls returned outside lecture hall).
  - No Electronic Recording of Lectures.
